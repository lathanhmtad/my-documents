I simply put an image in root directory folder. For example C:\Users\d0763\Documents\image. Now I want to access that image from browser through HTTP (like http://localhost:8080/...). My Tomcat server in running in Eclipse on 8080 port.this is my main project path D:\JavaBE\new-jdbc. Am I able to access that image through HTTP method?.
- No, you won’t be able to access the image through HTTP if it’s stored outside of your web application’s directory. You’ll need to move the image to a location within your web application’s directory, such as D:\JavaBE\new-jdbc\WebContent, and then you can access it through HTTP using a URL like http://localhost:8080/new-jdbc/image.
